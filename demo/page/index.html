A speedtest is a relay race: ping → download → upload, run against your own server — no third-party measurement servers. This demo measures the path between your browser and a Cloudflare Worker running @staffetta/server unmodified.

Demo transfers are capped at 32 MiB per request and a few seconds per phase — it shows the methodology, it is not meant to saturate a multi-gigabit link. MIT licensed.
