# staffetta — brand

Il marchio è **il cambio**: il momento del passaggio di testimone. La pillola grigia è il
corridore che lascia, quella arancione è il corridore che prende — client e server che si
scambiano i byte, sulla stessa traiettoria a −38°.

## File

| File | Uso |
| --- | --- |
| `mark.svg` | Marchio da solo, su fondo chiaro o scuro (il grigio è neutro per entrambi). |
| `lockup.svg` | Marchio + wordmark, per fondi chiari. |
| `lockup-dark.svg` | Marchio + wordmark, per fondi scuri. |
| `badge.svg` | Tessera arancione con il marchio in negativo — avatar GitHub, icona npm, social. |
| `favicon.svg` | Come il badge, con le pillole leggermente ingrandite per la resa a 16 px. |

## Palette

| Colore | Hex | Ruolo |
| --- | --- | --- |
| Arancione di gara | `#E8541F` | Il testimone che arriva; unico accento del sistema. |
| Grigio scia | `#8A929A` @ 55% | Il testimone che lascia, su fondo chiaro. |
| Grigio scia (dark) | `#5A636B` @ 80% | Il testimone che lascia, su fondo scuro. |
| Inchiostro | `#161A1E` / `#E8EAE6` | Wordmark su fondo chiaro / scuro. |

## Wordmark

`STAFFETTA` maiuscolo, peso 600, tracking ~0.2em, con la doppia **TT** in arancione: le due T
sono i due corridori del cambio. Font stack: Avenir Next → Futura → Century Gothic →
`ui-sans-serif`; nei lockup SVG il testo non è vettorializzato, quindi la resa segue i font di
sistema del viewer — per usi in cui serve fedeltà assoluta (stampa, immagini social)
convertire il testo in tracciati.
